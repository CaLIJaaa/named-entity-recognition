# Распознавание именованных сущностей

Курсовая работа по дисциплине «Машинное обучение в семантическом и сетевом анализе» (Финансовый университет, 2025/2026).

## О работе

Сравнение пяти подходов к задаче Named Entity Recognition (NER) — извлечения именованных сущностей (персон, организаций, локаций) из текста с помощью языковых моделей.

**Модели:**
- `O` baseline
- BiLSTM
- BiLSTM + CRF
- BERT (`bert-base-cased`)
- DistilBERT (`distilbert-base-cased`)

**Данные:**
- CoNLL-2003 — обучение и in-domain оценка (новостные тексты)
- WNUT 17 — out-of-domain оценка (соцсети), без дообучения

**Что исследуется:** качество моделей по entity-level F1, влияние domain shift при переносе на тексты другого стиля, эффект подбора гиперпараметров через Grid Search.

## Запуск

Работа выполнена в формате Jupyter notebook для Google Colab. Все зависимости зафиксированы в `requirements.txt`. Веса обученных моделей сохраняются на Google Drive.

Основной файл: `ner.ipynb`.
